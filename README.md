# ¿Que es un algoritmo?
Es una secuencia de pasos que describen la solución de un problema, por ejemplo:

podemos pensar, en **¿que pasos hay que seguir para ir a mercar?**

1. Levantarse
2. Bañarse
3. Organizarse
4. Desayunar
5. Tomar nota de lo que se va a comprar
6. Mirar que efectivo tengo
7. Tomar el bus
8. Coger los productos del supermecado
9. Hacer la fila en caja
10. Pagar
11. Regresar a casa
12. Organizar el mercado

Como vimos en el ejemplo anterior, podemos apreciar que para desarrollar un programa, debemos 
de describir paso a paso lo que debemos hacer para llegar a la solución.

# Condiciones del algoritmo
Para que la secuencia de instrucciones que desarrollemos sea considerada un algoritmo, esta debe:
- resolver un problema
- ser finito, es decir, debe tener un punto de finalizacion
- interactuar con datos externos

# Lenguajes de programación
Es una serie de reglas gramaticales que permiten darle instrucciones a una computadora, para ejecutar 
diferentes tareas. 

Existen diferentes tipos de lenguajes que son:
- **Interpretados:** Significa que cada una de las instrucciones que yo le diga, se van a ir traduciendo
  una por una a lenguaje de maquina para ir siendo ejecutadas, como por ejemplo python
- **Compilados:** Significa que todas las instrucciones son traducidas a lenguaje de maquina a la misma
  vez, para poder ser ejecutadas, como por ejemplo C++
  
Tambien podemos distinguir los lenguajes por:

- **Alto nivel:** que son los lenguajes muy parecidos al lenguaje natural de las personas para describir las instrucciones
- **Bajo nivel:** que son lenguajes que se asemejan mas a un lenguaje de maquina y son mas dificiles de escribir para un programador

# Algunos conceptos de los algoritmos
## Variables y constantes
- Las **variables** son espacios en memoria, en donde se almacena informacion temporalmente, de datos que utiliza el
  programa durante su ejecucion
- Las **constantes** son valores que no cambia su valor durante la ejecucion del programa

## Tipos de datos
En los lenguajes de programación, existen diferentes tipos de datos, que permiten reservar un determinado espacio de memoria,
entre los cuales se encuentra el entero, decimal, cadena de texto, booleano, fecha y binario, los cuales ayudan a describir 
mejor la solucion de un programa

## Operadores 
en programacion podemos utilizar diferentes operadores como:
 - **Aritmeticos:** que son utilizados para las operaciones matematicas basicas
 - **relacionales:** que son usados para evaluar expresiones condicionales, los cuales nos permiten obtener un *verdadero* o *falso*
 - **logicos:** que son usados para evaluar expresiones condicionales apartir de la union de varios de estos, para obtener un resultado final
